# React Keeper App (Part 3)

This project is a simple **Keeper (Notes) App** built with React.  
It focuses on managing dynamic lists using state, passing data via props, and handling user interactions like adding and deleting notes.

The project is inspired by Google Keep and is part of learning **React state management and component communication**.
